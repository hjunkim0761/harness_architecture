실행 주체별 역할

🔵 SCZ가 담당하는 것

  • 입력 수신, 인증 이후 보안 경계 내 처리
  • OCR 선처리와 실제 스킬 런타임 수행
  • ERP/그룹웨어/외부 API/RAG/DB 접근
  • 정책 확인, 승인 게이트, PII 탐지·범주화

🟣 Agent가 담당하는 것

  • 레시피 선택과 스킬 호출 순서 플래닝
  • 분기 판단과 사용자 대화 맥락 해석
  • 범주화·정책·검증 결과를 종합한 판단 보조
  • 최종 응답과 다음 액션 결정 후 자연어로 설명
  • 직접 시스템을 호출하지 않고 SCZ를 통해 실행
